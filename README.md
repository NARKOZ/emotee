# Dev QuickStart

### Ready

- Install [node.js](http://nodejs.org).
- Install CoffeeScript and UglifyJS with `npm install coffee-script uglify-js -g`.

### Set

- Clone Emotee and `cd` into it.

### Go

- Build it via `cake build`.
- Run continuous builds via `cake watch`.
